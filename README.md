# 成绩管家

> 记录每一步进步，见证每一次成长

一个简洁、有温度的成绩记录工具，专为家长设计，支持多档案管理，适合区分不同孩子或不同学习阶段。

## 核心功能

- 📝 **成绩录入** — 手动录入考试成绩（科目、分数、日期、考试类型）
- 📊 **趋势分析** — 趋势图表分析（单科趋势、总分变化、科目对比）
- 👤 **多档案管理** — 支持创建多个独立档案，每个档案拥有独立的考试记录空间
- 📤 **数据导入导出** — 支持 Excel 文件导入导出，方便数据备份和迁移
- 🔒 **本地存储** — 所有数据保存在浏览器本地，无需注册，隐私安全

## 技术栈

- 纯前端（HTML + CSS + JavaScript），无需后端服务
- [Chart.js](https://www.chartjs.org/) 图表库
- [SheetJS (xlsx)](https://sheetjs.com/) Excel 导入导出
- LocalStorage 本地存储
